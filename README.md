# telegram-manager
telegram manager bot

## Installation:
```bash
git clone "https://github.com/ali-shokoohi/telegram-manager.git"

cd telegram-manager/

mkdir logs/ || touch logs/bot.log

pip3 install -r requirements.txt #--user
```

## How to use:
```bash
cd bot/

echo "BOT_TOKEN" > TOKEN.txt #Input your bot's token here!

python3 bot.py &
```
